Italic
======

Source:

.. code-block:: rst

   This is normal. *This is italic.* This is normal again.

Result:

This is normal. *This is italic.* This is normal again.

References:

*  https://docutils.sourceforge.io/docs/ref/rst/restructuredtext.html#emphasis
